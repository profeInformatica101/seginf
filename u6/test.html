Seguridad en Redes Informáticas

  • IP Privada: 192.168.x.x, 10.x.x.x, 172.16.x.x - Para uso en redes locales.
  • IP Pública: Asignada por un proveedor de Internet para comunicación externa.
  • IP Estática: No cambia con el tiempo.
  • IP Dinámica: Asignada temporalmente por DHCP.

Endian Firewall permite segmentar la red en zonas:

  • GREEN: Red interna segura.
  • ORANGE: DMZ para servidores públicos.
  • RED: Conexión a Internet.
  • BLUE: Red WiFi controlada.

Para configurar Windows Server en la DMZ de Endian Firewall:

  1. Asignar IP fija dentro de la subred ORANGE (Ejemplo: 192.168.2.10).
  2. Configurar reglas de firewall para permitir tráfico HTTP/HTTPS.
  3. Habilitar seguridad adicional como IPSec o Firewall de Windows.
